# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-26

### Added
- Initial release of Mini SCADA HMI Dashboard
- Overview dashboard with machine status cards
- Machine detail view with event logs
- Runbooks for standard operating procedures
- Commissioning checklist with FAT/SAT sections
- Help/User Manual page
- Simulation mode for demo purposes
- Dark mode support
- Functional programming utilities (fp.js)
- Redux-like state management (state.js)
- Security utilities for XSS prevention
- Comprehensive test suite (tests.js)
- Hash-based SPA routing
- LocalStorage persistence
- JSON export/import for checklists
- Accessibility features (ARIA, screen reader support)

### Technical Details
- Zero-build architecture (no bundlers)
- CDN-only dependencies (Tailwind, Chart.js, Lucide, Tippy.js)
- Works via file:// protocol
- ~800 lines of code
- Browser-based testing
